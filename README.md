# 直播弹幕 TTS 软件

本软件功能如下：

- 监听B站直播间弹幕
- 过滤指定用户UID的弹幕
- 使用edge-tts将弹幕文本转换为语音
- 实时播放合成后的音频

## 下载安装

1. 运行`build_exe.bat`打包可执行文件`live_tts.exe`，文件在`dist/`。
2. 从ffmpeg官网下载ffmpeg并将ffplay.exe放置到`dist/`，或者说`live_tts.exe`所在目录下。
3. 双击运行`live_tts.exe`，填写的设置将自动保存到同目录下的`config.json`。

> 本软件遵循MIT协议。由于依赖第三方（B站）弹幕接口，该接口可能随时发生变更、限制或停止服务，届时本软件可能无法正常工作。对于因第三方接口变更、失效或停止服务导致的软件功能异常、不可用或数据丢失等情况，本软件作者不提供任何保证，也不承担由此产生的责任。

本软件功能主要有3部分：获取弹幕、发送弹幕和语音转文字（tts）。
当想获取弹幕时，直播间号、SESSDATA是必填的。
当想发送弹幕时，直播间号、bili_jct是必填的。
目标用户UID通常不用填，如果填写自己的UID，发送弹幕时可能导致双重播报。
